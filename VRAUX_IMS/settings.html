Status: Disabled

Save Auto Backup: Download the continuously updated backup file.

Note: Auto backup runs continuously when enabled and saves the latest data to memory. Click "Save Auto Backup" to download the current backup data as a file.

Restore Data: Restore data from a previously created backup file.

Important:
1. Backup files contain all your business data. Store them securely.
2. Please save backup files in the "backup" folder within your VRAUX_IMS directory.
3. Restoring data will overwrite all existing data.
4. Create the "backup" folder if it doesn't exist in your VRAUX_IMS directory.
